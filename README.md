# Who Are You?

#### WAU는 AI 필터를 이용하여 나만의 특별한 명함을 만들 수 있는 서비스입니다.

실제 다양한 서비스 환경에서 운영되기보다 특정 환경에서 시연을 목적으로 합니다.
